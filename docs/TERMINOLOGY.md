# Terminology

- Strategy prompt: the evolved text instruction used to generate a Java MicroRTS agent.
- Generated Java agent: source code produced from a strategy prompt before evaluation.
- Game-performance objective: NSGA-II objective derived from MicroRTS match metrics.
- Strategy-alignment objective: NSGA-II objective derived from LLM comparison of prompt and generated code.
- Mock mode: local deterministic pipeline mode for smoke tests and development.
