## Gap update rule

Close an implementation gap only when implementation, tests, configuration, artifacts, and affected canonical docs agree. A resolved architecture decision does not close its dependent implementation gaps. Record the validating test or schema version in the status text. Do not mark a gap closed from an old run artifact.
