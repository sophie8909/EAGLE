Experiment-specific assets, GUI-generated presets, and legacy strategy notes live here.

These files are not the source of default runtime values. Use
`configs/evolution/default.json` plus `eagle.config.EAConfig` for canonical
defaults, then save resolved run configs under `logs/eagle/<run>/`.

Some component JSON files in this directory are byte-identical historical
snapshots. Keep them unless a specific experiment lineage is intentionally
archived or removed.
